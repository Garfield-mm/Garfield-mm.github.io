4-Vue-组件

七月 01, 2022

1 概念

  • 组件是视图层的基本组成单元。
  • 组件是一个单独且可复用的功能模块的封装。
  • 组件上可以写属性，并对属性赋值。

2 注册

命名

  • kebab-case（短横线分隔命名）

    例如my-component-name.vue，引用时必须使用<my-component-name>

  • PascalCase（首字母大写命名）

    例如my-component-name.vue，可以使用<my-component-name>或<MyComponentName>

存放

在工程目录下新建components目录

全局注册

uni-app 支持配置全局组件，需在main.js 里进行全局注册，注册后就可在所有页面里使用该组件。

注意

  • Vue.component的第一个参数必须是静态的字符串。

  2. index.vue 里可直接使用组件

props

props 用于接收父组件的数据，可以是数组或对象。

选项 类型 说明
type String 、Number 、Boolean、 Array、 Object、 Date 、Function、 Symbol prop的类型
default any prop的默认值
required Boolean 是否必填项
validator Function 自定义验证函数
